# 🚀 MockMate – AI Mock Interview Platform (Backend)

Backend service for an AI-powered Mock Interview Platform built using **Java Spring Boot**, **Spring Security**, and **MongoDB**.

# 📌 Overview

MockMate Backend provides secure REST APIs for authentication, interview management, AI-powered interview feedback, resume analysis, and user profile management.

---

# ✨ Features

- User Authentication (JWT)
- Login & Registration
- Resume Upload & Analysis
- AI Generated Interview Questions
- AI Answer Evaluation
- Dashboard APIs
- Interview History
- Profile Management
- REST APIs
- Spring Security
