## 📖 About

MockMate Backend powers the AI Mock Interview Platform by handling user authentication, interview management, resume analysis, AI-powered feedback, and secure REST APIs.

The backend is built using **Spring Boot** following a layered architecture for scalability and maintainability.

---

## ✨ Features

- 🔐 JWT Authentication
- 👤 User Registration & Login
- 📄 Resume Upload & Analysis
- 🤖 AI Interview Question Generation
- 💬 AI Feedback System
- 📊 Dashboard APIs
- 📝 Interview History
- ⚡ RESTful APIs
- 🔒 Secure Endpoints
- 🌐 CORS Configuration

## ⚙ Installation

### Clone Repository

```bash
git clone https://github.com/harshagowda2409-bit/MockMate-your-interview-partner.git
```

### Navigate

```bash
cd MockMate-your-interview-partner
```

### Install Dependencies

```bash
mvn clean install
```

### Run

```bash
mvn spring-boot:run
```
